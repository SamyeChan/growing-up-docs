# Visual Studio Code

👉 [Visual Studio Code](https://code.visualstudio.com/)

- 简称：VS Code / VSC；
- 是一款免费开源的现代化轻量级代码编辑器；
- 支持几乎所有主流的开发语言的语法高亮、智能代码补全、自定义快捷键、括号匹配和颜色区分、代码片段、代码对比 Diff、GIT 命令等特性，支持插件扩展；
- 并针对网页开发和云端应用开发做了优化；
- 软件跨平台支持 Win、Mac 以及 Linux，运行流畅，可谓是微软的良心之作......

## wanna what

- “有多少实用的插件给我们带来开发便利和效率”
- “能否兼容我们的编程语言”
- “有没有定义或者方法跳转，让我在复杂的代码中定位到方法、变量和函数”
- “代码要看的舒服，代码高亮要优美好看，养眼”
- “有常用的快捷键，提高开发效率（最好让我不需要用鼠标）”

**总结一下其实就是“让我们写代码更快速，更愉悦，更舒服”**
