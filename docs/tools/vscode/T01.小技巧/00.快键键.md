# 【常用快捷键】

## 查看所有快捷键

先Ctrl + K，而后Ctrl + S

## 常用快捷键

```
Ctrl + Shift + P        打开控制面板
Ctrl + P                文件查找
Ctrl + F                文件中的字符查找
Ctrl + `                打开命令行终端
Ctrl + Shift + N        打开一个新编辑器
Ctrl + Shift + W        关闭一个编辑器
Shift + Alt + \         拆分编辑器？
Ctrl + B                快速打开和关闭侧边栏
Ctrl + Shift + F        代码格式化
Ctrl + Shift + X        调出插件窗口
Ctrl + K 后 Ctrl + X    清除空格（一般设置自动清除了的）
Ctrl + Shift + [        代码折叠
Ctrl + Shift + ]        代码展开
Shift + Alt + ↑/↓       向上/下复制行
Ctrl + G                跳转到特定行数
Ctrl + Shift + O        文件中跳转特定符号
```

- 自动清除空格： settings.json 中设置 `"files.trimTrailingWhitespace": true`；
