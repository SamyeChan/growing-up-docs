自我评价作为自我认知的重要部分，是一个人从自身的性格、爱好、特长、思维方式、意志品质、知识结构与素养等方面出发，对自己的身心状况、能力、特点、社会处境以及与他人和社会关系的一种动态认识。
 
从本源来说，自我评价与童年的成长息息相关。在我们有了一定的认知意识之后，人的三个“我”：本我、自我、超我就会逐渐成长。超我是代表理想化的自己，本我是人的本能，而自我则是在两者之间协调的部分。正如心理学家库利对自我的定义：“对每个人来说，他人都是一面镜子，个人通过社会交往了解到别人对自己的看法，从而形成自己的自我”。
 
在童年阶段，父母和师长是以激励赞赏还是压制惩罚的方式去管教孩子，往往会给孩子造成不同的影响，进而影响其自我认知。对于童年期心智不成熟的孩子来讲，如果感到恐惧和不安，那么他往往无法获得自信。绘本大师莫里斯·桑达克就曾谈到：每个人都不要轻视童年时代的恐惧与不安，它们将伴随人的一生。

真实的情况是，大部分孩子都会想获得来自父母师长的认同和赞美，甚至会想很多办法来获取关注。但是一旦因为某些客观原因或者父母师长的责怪而得不到认同时，自尊心强的孩子就容易产生内疚和负罪的心理，认为是自己做错了，在多次的事件积累之后，就会导致自我评价低的内心感受。并且，有些孩子还容易忽略客观存在的事实和原因，一旦失利就把不好的结果都归结到自己身上，认为是自己不努力或者自己不够好造成的，通过自责将这种自我评价低的感受再度不断强化。

自我评价低都有怎样表现？
 
正如梭罗所说：一个人怎么看待自己，决定了此人的命运，指向了他的归宿。自我评价低的人很容易产生自我否定，错误的把“我做不到某些事情”等同于“我不够好”，直接全盘否定了自己。
 
表现上看，自我评价低的人往往更渴望得到社会的认同，因为他们自己并不认同自己，而是会羡慕一些活的洒脱、不在意外界评论的人，可同时自己又没有勇气做出改变。从日常的行事来说，低评价者容易感觉担当不起别人的称赞而妄自菲薄，非常没有自信心。在职场中，上述两种表现综合起来，则表现的抗压能力极低，显得非常缺乏领导力。
 
而由于不自信，自我评价低的人还很怕与人发生争执。因为内心的怯懦，让他们会没有勇气拒绝别人，因为害怕会伤害别人。这种害怕争执的特点，容易让他们在亲密关系中没有坚定的信心，当机会或者心仪异性出现的时候，自我评价低者往往会觉得自己配不上对方而犹豫退缩；而一旦遇到争执又会充满负能量怕被抛弃，因此会刻意迁就和妥协，严重者甚至不能与人建立长期的亲密关系回归到整个行动机制来看，自我评价低的后果就是不自信，习惯性的做负面诠释，习惯性的自我否定，甚至将责任都归结到自己身上。
 
如何改善自我评价低？
 
上述的这些表现，其实我们在生活中都会常常碰到，而来自调研的数据也显示，自我评价低的人口并不是少数。前几年，杭州市曾在辖属某区的2000名学生中做了一个调查问学生：“碰到困难时，你觉得自己无用吗？”14.1%的小学生和23.1%的中学生选择了“是”，达到了近两成的比例。可以说在当今的社会人口尤其是年轻人当中，自我评价低已经是很多人的一种隐性心理疾病。
 
要改变这一现状，首先从自我认知的角度，要正确的认识自己，这包括对自己的看法以及社会给自己灌输的观念两个方面。对自己的看法应该要综合评估，而不是仅仅查看自己某一时负面情绪带来的感受；对于周围人带来的价值观，也应该要合理分析，在分析后形成“自我观”，而不是被某种思想“洗脑”。
 
然后是学会树立自己的自信，并将其作为核心信念不断强化。面对批判声音如比如“太笨了”“很差劲”“不够好”时，要有自己正确的判断，该改善的改善，该坚持应该坚持，切不可将外界负面声音带来的情绪内化成自我伤害，甚至自我攻击的行为，而应该多想事情的积极方面和解决方法。当你面对任何攻击，不够自信的时候，唯一的解决方式，正如《野兽国》的主人公马克斯所做的，与它们无所畏惧地对视，并向它们证明自己更强大。
 
最后则是要行动起来。正如哲学家马洛所说，荣誉要靠我们用行动去争取。因为只有行动可以改变现状。并且，在行动过程中要给自己树立目标，再将目标分解成一个个小目标，接着一步步努力完成每一个小目标。因为任何一个小目标的达成，都会令我们的自尊和自信心得到改善。
 
当然，如果在改善的过程中再成熟一点，可以尝试在人际关系中试着不要刻意讨好别人。不要总是抱着“如果我说出自己的真实想法，我们会不愉快”之类的想法，而是尝试真实的表达自己，并在真实和真诚中建立良好的人际关系。甚至有时候学会向别人要支持，这并不是没有自尊的表现，相反这是一个人成熟，敢于放下面子的表现。
 
想起聪慧的张爱玲曾感叹：最恨有天才的女孩子早结婚。——其实，她所说的并不是女孩子不应该在年轻时结婚，而是每个个体不应该在社会评价体系以及别人给的价值观里成为一个自我评价低的人，进而成为“活在别人眼里”的人生loser。
