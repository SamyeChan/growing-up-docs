# 【Axios】

axios 是一个基于 Promise 用于浏览器和 nodejs 的 HTTP 客户端，它可以帮你：

- 从浏览器中创建 XMLHttpRequest
- 从 node.js 发出 http 请求
- 支持 Promise API
- 拦截请求和响应
- 转换请求和响应数据
- 取消请求
- 自动转换 JSON 数据
- 客户端支持防止 CSRF/XSRF
