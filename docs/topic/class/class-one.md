## 组件通信

父 -> 子

- props（子）
- 引用 refs（父）

子 -> 父

- $emit('add', good)（子）
- @add="cartAdd($event)"

兄弟组件

- $parent -> $on / $emit

祖先和后代之间

祖先 -> 后代

- provide / inject （实现祖先给后代传值）- 主要为高阶插件/组件库提供用例，并不推荐直接用于应用程序代码中 -> 多见于开源组件库；

后代 -> 祖先

dispatch 后代给祖先传值
