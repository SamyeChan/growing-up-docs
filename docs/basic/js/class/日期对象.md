# 日期对象

- 一个和日期有关的对象，用来处理日期和时间，是由js已经提供给我们的 - 内置对象；

## 内置对象

- Array
- Object
- Date
- ...

### Date()

```js
var date = Date();
console.log(date);

/* Mon Jul 29 2019 10:39:13 GMT+0800 (中国标准时间)
 * 星期一 7月 20日 2019年 时间 市区
 */

console.log(typeof date); // string
```
- Date是一个函数，但如果直接通过 () 进行调用，返回一个字符串；

### new Date()

```js
var date = new Date();
console.log(date);
console.log(typeof date); // object
```
- 通过 new 调用的 Date 得到一个对象，而非字符串；
- 日期对象下有日期和时间的一些方法；

## 参数

- 函数调用的时候是可以传入参数的 - **创建日期对象时，传入参数不同，就会有不同的效果**；
- 若无参数，由“当前系统时间”作为参数，返回一个日期对象；
- 若有参数，根据“自提供参数”，返回一个日期对象；

1. `value` - 代表自1970年1月1日00:00:00（世界标准时间）起经过的毫秒数；
  - 代指毫秒数（ms）；
  - 可正可负；
  - 历史：UNIX（贝尔实验室，1969），时间用32位带符数字最多可表示68年；
2. `dateString` - 表示日期字符串

  字符串形式：
  ```js
  var date = new Date('2019-7');
  console.log(date);

  // Mon Jul 01 2019 08:00:00 GMT+0800 (中国标准时间)
  ```
  - `yyyy-mm-dd`；
  - 至少两个参数；
  - 一但输入的日期不合法 - Invalid Date；

  ```js
  var date = new Date(2035, 2);
  console.log(date);

  // Thu Mar 01 2035 00:00:00 GMT+0800 (中国标准时间)
  ```
  - 注意，月份从 0 开始计算；
  - 必填2项：year、month；
  ```js
  var date = new Date(2035, 15);
  console.log(date);

  // Tue Apr 01 2036 00:00:00 GMT+0800 (中国标准时间)
  // 年份推只下一年了
  ```
  - 若传入值不符合时间标准，回进行**自动计算**；
  - 若使用的是字符串形式，就不会进行**自动计算**；

3. year - 代表年份的整数值；
4. month - 0～11，0代表1月，11代表12月；
5. day - 代表一月中的某一天，从1开始；
6. hour - 代表一天中的小时数，从0开始；
7. minute - 分钟数，从0开始；
8. second - 秒数，从0开始；
9. millisecond - 毫秒数，从0开始；

#### 注意

1. 若无参数，会根据系统设置的当前时间来创建一个 Date 对象；
2. 至少提供2个参数，其余参数均默认为1或0；
3. 通过函数直接调用返回的是tring，通过 new 调用的是 Object；
4. 通过第三种方式设置值，一旦超过就会进行自动计算；

### 日期对象相关方法

```js
var date = new Date(1996, 2, 5, 22, 0, 0);
console.log(date);

// Tue Mar 05 1996 22:00:00 GMT+0800 (中国标准时间)
```

#### getFullYear()

返回指定日期对象的年份：

```js
date.getFullYear();

// 1996
```

#### getMonth()

返回指定日期对象的月份：

```js
date.getMonth() + 1;

// 3
```

#### getDate()

返回指定日期对象中的第几天：

```js
date.getDate();

// 5
```

#### getDay()

返回指定日期对象星期中的第几天：

```js
date.getDay();

// 2
```

#### getHours()

返回指定日期对象的小时数：

```js
date.getHours();

// 22
```

#### getMinutes()

返回指定日期对象的分钟数：

```js
date.getMinutes();

// 0
```

#### getSeconds()

返回指定日期对象的秒钟数：

```js
date.getSeconds();

// 0
```

#### getMilliseconds()

返回指定日期对象的毫秒数：

```js
date.getMilliseconds();

// 0
```
- 如果想获取一个正常的数值，那么在设定时就不应该设置 >1000 的值；

#### getTime()

返回从1970年01月01日至该日期对象经过了多少毫秒数，值会有正负：

```js
date.getTime();

// 826034400000
```

### 设置日期对象具体时间的方法

```js
var date = new Date(1996, 2, 5, 22, 0, 0);
console.log(date);

// Tue Mar 05 1996 22:00:00 GMT+0800 (中国标准时间)
```

#### setFullYear()

为指定日期对象设置完整年份：

```js
date.setFullYear(2019);
console.log(date);

// Tue Mar 05 2019 22:00:00 GMT+0800 (中国标准时间)
```

#### setMonth()

为指定日期对象设置月份：

```js
date.setMonth(4);
console.log(date);

// Sun May 05 2019 22:00:00 GMT+0800 (中国标准时间)
```

#### setDate()

为指定日期对象设置天数：

```js
date.setDate(3);
console.log(date);

// Fri May 03 2019 22:00:00 GMT+0800 (中国标准时间)
```

#### setHours()

为指定日期对象设置小时数：

```js
date.setHours(11);
console.log(date);

// Fri May 03 2019 11:00:00 GMT+0800 (中国标准时间)
```

#### setMinutes()

为指定日期对象设置分钟数：

```js
date.setMinutes(22);
console.log(date);

// Fri May 03 2019 11:22:00 GMT+0800 (中国标准时间)
```

#### setSeconds()

为指定日期对象设置秒钟数：

```js
date.setSeconds(33);
console.log(date);

// Fri May 03 2019 11:22:33 GMT+0800 (中国标准时间)
```

#### setMilliseconds()

为指定日期对象设置毫秒数：

```js
date.setMilliseconds(666);
console.log(date);

// Fri May 03 2019 11:22:33 GMT+0800 (中国标准时间) ????????
```

#### setTime()

通过毫秒数来设置日期对象时间，以1970年1月1日为时间起点:

```js
date.setTime();
console.log(date);

// ??????????
```
