#### 函数数组

功能抽离, 把每一块的功能单独的抽离处理，封装成单独的一个处理函数
1. 添加事件
2. 点击事件只做一件事：修改数据状态(sortState 排序状态，filterState筛选的状态)
3. createData 根据状态码对数据进行操作
4. render 根据数据渲染视图
数据驱动视图, 不要去操作视图，而是操作数据，然后通过数据在渲染视图 
!! 不要直接修改页面内容，而是修改数据，修改完数据之后，通过数据去同步我们的页面
